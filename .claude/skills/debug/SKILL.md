name: debug-skill
description: Help debug code 
---

1. You come up with hypotheses
2. Add logging to the code via fetch to a web server instead of console.log
3. Start the server and make sure the server is running
4. You stop and tell me the steps to reproduce the issue 
5. I confirm the issue was reproduced
6. You review the logs and confirm or reject the hypotheses then update the instrumentation as new information becomes available
7. Remove logs after I've confirmed you've fixed it
